# Hotel Booking Analytics

## 📌 Project Overview

This project analyzes hotel booking data using MySQL, Python and Tableau to identify booking trends, customer behavior, revenue drivers and cancellation patterns.

---

## 🎯 Business Problem

Hotel management wants to understand customer booking behavior, reduce cancellations and maximize revenue.

---

## 🎯 Objectives

- Analyze booking trends
- Analyze revenue
- Study customer behavior
- Identify cancellation patterns
- Provide business recommendations

---

## 🛠 Tools Used

| Tool | Purpose |
|------|---------|
| MySQL | Data Cleaning & Analysis |
| Python | EDA & Statistical Analysis |
| Tableau | Dashboard |
| GitHub | Version Control |
